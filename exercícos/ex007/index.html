Texto Marcado

Podemos criar também um texto marcado usando a tag mark
Para trocar a cor da marcação, é só utilizar a extensão "styles", dentro da tag mark, procurar por background color e procurar a cor desejada
Ex.:Marcação de outra cor
Caso tenha muitas tags marks, não precisaconfiguraar todas, é só no início do programa, colocar o seletor :
<style>
(tag desejada, nesse caso, mark) {
background color : cor desejada
}

Texto Deletado


Podemos marcar um texto excluído para mostrar que este deve ser lido, mas não considerado, usando o atalho de pesquisa e a tag <del>.


Texto inserido

Há ainda a tag <u> que deixa o texto com o aspecto sublinhado
