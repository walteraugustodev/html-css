Trabalhando com listas

As listas ordenadas são aquelas que a ordem faz diferença, como uma receita e as não ordenadas, as que a ordem não faz diferença, como uma lista de compras

Listas Ordenadas

Para evitar problemas de numeração de lista, por exemplo, se vpcê lembra de um item depois da lista já grande, seria necessário mudar toda numeração, (do quinto pro sexto, do sexto pro sétimo, etc) para evitar esses problemas existe a tag<ol>, que enumera a lista, e a tag <li>, para indicar itens de lista
  1. Acordar
  2. Ligar pro João
  3. Tomar café
  4. Escovar os dentes
  5. estudar
    6. tambem pode usar o parametro type, na tag ol, que muda o jeito que a lista será ordenada, numeros, letras...

Listas não ordenadas

Se usa a tag ul
  • Tomate
  • Alface
  • Pão
  • Leite
Também existem parametros pra as duas listas, que podem ser usadas pra alterar o símbolo que marca a lista
